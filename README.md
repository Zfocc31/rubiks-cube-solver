# Rubik’s Cube Solver with OpenCV Scanner

A C++ based Rubik’s Cube solver that combines
computer vision (OpenCV) for cube state detection
with a pattern-database-based solving algorithm.

---

## Features
- Live Rubik’s Cube scanning using OpenCV
- Color detection and classification
- Pattern Database based solver (depth 9)
- Modular C++ design with CMake
- Fast and memory-efficient solving pipeline

---

## Tech Stack
- C++
- OpenCV
- CMake

---

## Project Structure
- `Scanner/` – Camera input and color classification
- `Solver/` – Cube solving logic using pattern databases
- `Model/` – Cube representation and transformations
- `Databases/` – Precomputed pattern databases
